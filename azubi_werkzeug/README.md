# Azubi Werkzeug Tracker Add-on

Dieses Add-on integriert den Azubi Werkzeug Tracker in Home Assistant.

## Installation

### Lokales Add-on
1. Erstellen Sie einen Ordner namens `azubi_werkzeug` in Ihrem Home Assistant `addons`-Verzeichnis (z.B. `/addons/azubi_werkzeug`).
2. Kopieren Sie alle Dateien aus diesem Repository in diesen Ordner.
3. Starten Sie Home Assistant neu oder gehen Sie zu **Einstellungen -> Add-ons -> Add-on Store** und klicken Sie auf **Überprüfen auf Updates** (im Menü oben rechts).
4. Das "Azubi Werkzeug Tracker" Add-on sollte nun unter "Lokale Add-ons" erscheinen.
5. Installieren und starten Sie das Add-on.

## Konfiguration

Das Add-on benötigt keine spezielle Konfiguration. Der Datenbankpfad wird automatisch verwaltet und die Daten bleiben auch nach Neustarts erhalten (im `/data`-Verzeichnis des Add-ons).

## Nutzung

Klicken Sie nach dem Start auf "Web-Benutzeroberfläche öffnen", um die Anwendung zu nutzen.
